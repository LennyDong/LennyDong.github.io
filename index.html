Projects

Memokee

Never forget another password, again.

Memokee is a simple password storage web application I built with my friend using Node.js, Express.js and Firebase. The insipiration of Memokee came from my very own frustration with my inability to match the correct username & password pair to their corresponding service. With all the varying password requirements different websites and forums have, I have probably created a dozen diffrent versions of the same password at this point. By memorizing just one password, the user can access all of their username & password pairs stored using Firebase's secure database. This is still a work in progress

Gitlet

Because I needed to learn how Git works.

Gitlet is a VCS that in practice is very similar to that of Git's, but was built using my own data structures and design. Ironically, I went into this school project with very limited knowledge of how to use git, but struggling through this project has solidified my understanding of it.

Customer Development (Oct 2015 - Now)

Ava

Ava stands for Auditory Visual Aid. It is an Oakland-based start-up that aims to bridge the communication gap between the deaf/Hard of Hearing and the hearing. I interact with our testers and customers, to make sure their feedbacks are reflected in our product. Vice Chair for 8th Korea America Student Conference (Aug 2014 - Jul 2015)

International Student Conference

Korea America Student Conference(KASC) brings together a total of 50 college students from both Korea and America to participate in the month long conference during which they travel and particiapte in various events across the US/South Korea together. I was elected to become one of the ten student leaders to organize the entire conference. We were charged with everything ranging from contacting host universities and transportation logistics, to actually leading the month long conference in July. This remains to be one of my most rewarding experiences - I learned a great deal about my leadership style, and the qualities that are essential to hold such a position.
